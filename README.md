# Object-Oriented Programming in C++

This repository contains my implementations of Object-Oriented Programming concepts in C++. The programs are based on university practical exercises and cover fundamental as well as advanced OOP topics.

---

## Topics Covered

- Classes and Objects
- Constructors and Destructors
- Encapsulation
- Friend Functions
- Operator Overloading
- Function Overloading
- Inheritance
- Runtime Polymorphism
- Abstract Classes
- Virtual Functions
- File Handling
- Exception Handling
- Templates
- Namespaces

## Compilation and Execution

Compile any program using:

```bash
g++ filename.cpp -o output
./output
```

Example:

```bash
g++ 01_student_data_class.cpp -o student
./student
```
